# Notes app

This a mini notes app which allows the user to write,read and edit their notes, along with change the font of the notes and save it into a file.

Features:

- File storage
- Error handling
- Create, view, edit and manage notes easily 
- Save notes to a file
- Delete notes when no longer needed 

What I learnt:

- Reading/writing files 
- Variables 
- Defining functions 
- Conditional loops
